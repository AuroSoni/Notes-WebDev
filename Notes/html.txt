1. basic setup:
<!DOCTYPE html>
<html>
  <head>
    <title>My Page</title>
  </head>
  <body>
    Hello world
  </body>
</html>
2. Headings:
  <h1></h1>                 //largest
  ...
  <h6></h6>                 //smallest
3. Lists:
  i)Ordered lists:
    <ol>
    <li>text
    <li>text
    </ol>
  ii)Unordered lists:
    <ul>
    <li>text
    <li>text
    </ul>
4. Images:
  <img src="[path]filename">      //full resolution img displayed
  <img src="..." height="no" width="no">    //no in pixels
  <img src="..." height="no">               //proportionate img resolution
  <img src="..." height="percent">          //responsive to browser size
5. Tables:
  <table>
    <tr>
      <th>text</th>
      ...more ths          //table headings
    </tr>
    <tr>
      <td>text</td>
      ...more tds         //table data
    </tr>
  </table>
6. forms:
  <form>
    <input type="type_name" placeholder="" name="">
    <button>Submit!</button>
  </form>
